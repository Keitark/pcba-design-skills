# Cost-aware substitution review

Derive the real electrical, timing, power, package, thermal, mechanical,
programming, lifecycle, and quantity requirements before searching. Use the
official datasheet for the exact orderable suffix and current JLCPCB/LCSC
evidence. Calculate total assembled-order impact including MOQ, setup, feeder,
extended-part, and attrition costs.

Compare every physical pin's role, direction, active level, voltage domain,
pull, safe state, and static tie. Compare the exact manufacturer package drawing
and footprint dimensions. Similar capacity, family, pin count, or package name
is insufficient.

Classify a candidate as exact drop-in, bounded design change, redesign required,
or reject. Obtain explicit approval before changing a critical match. Any
selected package/electrical change invalidates the source design and complete
manufacturing release; repeat the complete BOM and placement review after a new
upload.
